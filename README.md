・ゲーム概要  
オンライン対戦ローグライクゲームです。  
探索パートで落ちているアイテムを集めたり、敵を倒してレベルを上げたりして自キャラを強くし、そのステータスを用いて対戦パートで敵と戦いましょう。  
4人までの同時対戦が可能です。
